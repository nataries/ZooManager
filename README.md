#Zoo Manager - WPF приложение для управления зоопарками
  Zoo Manager - это WPF приложение, работающее совместно с SQL Server 2022 Expression, которое позволяет управлять зоопарками, животными и связью "какое животное обитает в определённом зоопарке".
  Структура базы данных представляет собой 3 таблицы: Zoos - города, в которых есть зоопарки, Animals - все животные, не привязанные к определенному зоопарку и ZooAnimal - таблица, отображающая, в каком зоопарке обитает 
  определенное животное.
  Интерфейс приложения состоит из трех ListBox, каждый из которых выполняет отдельные функции. 
  Zoo List - первый ListBox, в котором представлен список городов, где есть зоопарки. К этому ListBox относятся 
  несколько кнопок. Add Zoo - добавляет введенный в TextBox город в Zoo list. Update Zoo - обновляет название
  зоопарка, заменяя на название, введённое в TextBox. Delete Zoo - удаляет выбранный город из списка зоопарков.
  Associated Animals - второй ListBox, отображающий животных, которые есть в зоопарке того города, который выбран
  в ListBox Zoo List. Кнопка Remove Animal удаляет выбранного животного из зоопарка, выбранного в первом ListBox.
  All Animals - третий ListBox отображает список всех животных. Кнопки этого ListBox выполняют различные функции,
  расширяя функционал приложения. Add Animal To Zoo - выбираем город из первого ListBox, выбираем животного
  из третьего ListBox, нажимаем на эту кнопку, и выбранное животное добавляется в зоопарк. 
  Add Animal Zoo - добавляет введенное в TextBox животное в ListBox All Animals.Update Animal - обновляет 
  название выбранного животного из третьего ListBox, заменяя его на название, введённое в TextBox. 
  Delete Animal -удаляет выбранного животного из ListBox All Animals.
